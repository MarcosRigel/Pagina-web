Sobre cada tag aprendida nas aulas no Módulo de Primeiros passos com HTML

  • <p> - é usada para criar um parágrafo de texto
  • <h1> ate o <h6> - é utilizado para criar textos de diferentes tamanhos
  • <i> - tag utilizada para deixar o texto em itálico
  • <strong> - para deixar o texto em negrito
  • <ol> - para criar uma lista ordenada
  • <ul> - para criar uma lista nao ordenada ou tópicos
  • <a> - para criar uma ancora que leva a navegação para algum outro lugar da página ou para outro site
  • <hr> - para criar uma linha depois do parágrafo
  • <blockquote> - para criar um espaçamento no parágrafo
